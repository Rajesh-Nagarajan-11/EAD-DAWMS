# 🖥️ EAD-DAWMS – IT Asset Warranty Management System
## 📦 Installation

To install and set up the project:

```bash
git clone "https://github.com/Rajesh-Nagarajan-11/EAD-DAWMS"
cd EAD-DAWMS
npm install
```


## 🚀 Run the Project

Start the development server with:

```bash
npm run dev
```

http://localhost:3000
